# metric-tracker

> 🇺🇸 [English README](./README.md)

**지표 설계·추적·리뷰 엔진. OKR·KPI·핵심지표를 구조화하고 주기적 리뷰를 프로토콜화.**

## 목표
허점 지표에서 벗어나 성과 중심의 측정으로 전환합니다. 의미 있는 지표 설계, 추적 시스템 구축, 구조화된 리뷰 실행을 통해 원시 데이터를 실행 가능한 통찰력으로 변환하고 사업 의사결정을 주도합니다.

## 주요 기능
- **OKR/KPI 설계 프레임**: SMART 기준에 따른 목표와 핵심 결과 정렬, 팀 간 목표 계층화
- **구조화된 추적 시스템**: 지표 정의, 데이터 소스, 수집 일정, 기준선 설정
- **주기적 리뷰 프로토콜**: PDCA 기반 점검 사이클, 편차분석, 시정조치
- **대시보드 템플릿**: 실시간 지표 현황 파악, 추이분석, 예측
- **액션 루프 통합**: 지표 인사이트를 전략 과제 및 자원배분과 연결
